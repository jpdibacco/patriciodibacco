Technical aspect:

This project is divided in several parts:

From one side there is the hardware part, which is basically an ESP32 module connected to several sensors such us:

Flex sensor, gyroscope and a light sensor.

The device was built in the form of a doll and we added also some outputs such a led lights and a vibrator motor.

The other hardware built by us was a glove with a capacitor sensor and a vibrator motor on it. We used an ESP8266 module to receive the signals from the capacitor sensor and send some to the vibrator motor.

We also used a router to facilitate the wifi connection between devices and laptops.

From the other side, there is the software parts:

  • We used node-red to connect all the devices between each others.
  • We used the Arduino software to code both devices: the doll and the glove.
  • We used Unity to create a virtual environment that would react according to the parameters of the doll and the glove.
  • We used Purr-Data (PD-L2ork extension of Pure Data software) to create a Drone-Music environment that would react according to the parameters of the glove and the doll.
